üöÄ Quick Start Guide

Follow these steps to set up your training system from scratch:

  1. Create Organizational Structure Start by creating Units (e.g., ELS, AMT, UNIT-3), then add Lines within each Unit (e.g., RED LINE, TATA LINE), and finally add Operations to each Line.
  2. Upload Training Videos Navigate to Training Videos section, select the operation, upload video file, add title and description, set duration, and mark it as active.
  3. Create MCQ Tests For each video, create an MCQ test with passing score (e.g., 70%), time limit, and add multiple questions with 4 options each.
  4. Add Employee Profiles Create user accounts and link them to employee profiles with employee ID, plant, unit, and department information.
  5. Monitor Progress Use the Dashboard to track video completions, test scores, and generate comprehensive reports.

üí° Pro Tip

Set up the organizational hierarchy (Unit ‚Üí Line ‚Üí Operation) completely before adding videos. This makes content organization much easier and prevents having to reorganize later.

üìö Content Management

üè¢ Managing Units, Lines & Operations

The organizational hierarchy is the foundation of your training system:

Units

Top-level organizational divisions like ELS, AMT, UNIT-3. Each unit can contain multiple production lines.

Lines

Production lines within a unit like RED LINE, TATA LINE, MAHINDRA LINE. Each line contains specific operations.

Operations

Specific work operations like OP-100 PLUGGING, OP-123 OPV. Each operation can have training videos and tests.

üé• Training Videos

  1. Select Operation Choose the operation this video belongs to from the dropdown menu.
  2. Add Video Details Provide a clear title, detailed description, and upload the video file. Optionally add a thumbnail image.
  3. Set Properties Specify duration in seconds, mark if video has voice narration or callouts, set display order, and activate it.
  4. Create Associated Test Click on the video after saving and add an MCQ test directly from the inline form at the bottom.

‚ö†Ô∏è Important

Always create an MCQ test for each training video to assess employee understanding. Videos without tests will appear in the "Videos Needing Attention" section of the dashboard.

üìù MCQ Tests & Questions

Create comprehensive assessments for each training video:

  1. Test Configuration Set test title, description, passing score percentage (recommended: 70%), and time limit in minutes (recommended: 30).
  2. Add Questions Use the inline question form to add multiple questions. Each question needs:
    • Question text
    • Four options (A, B, C, D)
    • Correct answer selection
    • Marks for the question
    • Question order number
    • Optional explanation for learning
  3. Review & Activate Ensure all questions are properly configured and mark the test as active.

üí° Best Practices for Questions

Create 10-15 questions per test, mix easy and challenging questions, include practical scenarios from the actual operation, and always provide explanations for correct answers to enhance learning.

üë• Employee Management

Creating Employee Profiles

  1. Create User Account First create a Django user account with username, password, first name, and last name.
  2. Add Employee Profile Link the user to an employee profile with unique employee ID, biometric ID (optional), plant, unit, and department.
  3. Assign Permissions Regular employees don't need special permissions. Create separate admin accounts for managers.

üí° Employee ID Format

Use a consistent format for employee IDs (e.g., EMP001, EMP002) to make searching and filtering easier in reports.

Tracking Employee Progress

Monitor individual and team performance through:

Video Completions

View completion percentage, number of accesses, and last watched date for each video.

Test Attempts

Track test scores, pass/fail status, time taken, and number of attempts per employee.

Login Sessions

Monitor when employees log in, session duration, and total training time spent.

üìä Reports & Analytics

Dashboard Overview

The main dashboard provides real-time insights into:

  • Summary Statistics: Total employees, videos, tests, completion rates, and pass rates
  • Top Performers: Employees with most video completions and test passes
  • Videos Needing Attention: Videos with low completion or missing tests
  • Problem Tests: Tests with pass rates below 70%
  • Unit Progress: Training statistics organized by unit
  • Recent Activity: Latest video completions and test attempts

All admin lists support powerful filtering:

‚ú® Best Practices

Content Quality

  • Keep training videos between 5-15 minutes for optimal attention
  • Use clear titles that describe the operation (e.g., "OP-100: Wire Plugging Process")
  • Add detailed descriptions explaining what employees will learn
  • Upload high-quality video files (1080p recommended)
  • Create custom thumbnails to make videos more appealing
  • Mark critical CTQ (Critical to Quality) operations clearly

Test Design

  • Create 10-15 questions per test for comprehensive coverage
  • Mix different difficulty levels (easy, medium, hard)
  • Focus on practical scenarios employees will face
  • Set passing score at 70% minimum for critical operations
  • Allow 2-3 minutes per question when setting time limits
  • Always provide explanations to reinforce learning
  • Review and update questions based on common mistakes

Employee Training

  • Assign training based on employee's unit and department
  • Monitor progress weekly using dashboard reports
  • Follow up with employees who have low completion rates
  • Recognize top performers to encourage others
  • Review test results to identify knowledge gaps
  • Update content based on employee feedback

‚ö†Ô∏è Data Backup

Regularly export reports and backup your database. Keep copies of original video files in a separate location. Schedule automated backups of the Django database.
